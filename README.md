# React Personal Website

A modern, responsive personal portfolio website built with React.js and D3.js, showcasing interactive data visualizations and professional projects.

## Built With

- **React 18** - Frontend framework
- **D3.js v7** - Data visualization library
- **React Bootstrap** - UI components and responsive grid
- **Observable Runtime** - For embedding Observable notebooks
- **CSS3** - Custom styling and animations

## Project Structure

```
src/
├── components/
│   ├── chart/          # Data visualization components
│   └── ui/             # UI components
├── data/               # CSV and JSON data files
├── images/             # Static images
└── static/             # PDF and other static assets
```

## Getting Started

### Prerequisites

- Node.js
- npm

### Installation

1. Clone the repository
```bash
git clone https://github.com/haoshuai999/me.git
cd me
```

2. Install dependencies
```bash
npm install
```

3. Start the development server
```bash
npm start
```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

## Deployment

The site is deployed using GitHub Pages:

```bash
npm run deploy
```

This builds the project and pushes to the `gh-pages` branch.
